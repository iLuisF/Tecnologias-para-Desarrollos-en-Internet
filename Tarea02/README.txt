Se desarrollo en Netbeans con Spring, Mysql y JDBC. El servidor es glassfish.

La base de datos(DDL) que se uso para implementar esta dentro del directorio sql.
El proyecto de Netbeans se encuentra dentro de la carpeta src.

Para compilar solo se debe importar a Netbeans y correr mendiante este, el formulario
con las validaciones se encuentra en form.htm

------

Aclaraciones

Los campos del formulario se validan, pero al ser "campos constantes" el único campo
que se le asigna un valor es el de nombre los demás estan como defecto para una pelicula.
Es decir, el formulario es para confirmar que dadado una pelicula con sus datos se guarde en
la base de datos el boleto de esta pelicula, y el formulario ya tiene algunos campos
predefinidos y otros se tienen que llenar. 

El sistema funciona correctamente.
